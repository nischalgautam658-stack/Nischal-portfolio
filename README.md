## About
This repository contains the source code for Nischal Gautam's personal portfolio website. The site highlights his skills as a developer, designer, and digital enthusiast, emphasizing creativity, interactivity, and responsiveness.

---

## Features
- Interactive hero section with animated text
- Core skills showcased through a tab-based layout
- About section detailing experience, innovation, and passion for technology
- Contact form for inquiries and collaboration
- Social media links integrated:
  - Instagram
  - Facebook
  - GitHub
- Responsive and mobile-friendly design
- Modern CSS animations and particle effects

---

## Tech Stack
- **HTML5** – Markup structure of the site
- **CSS3** – Styling and animations
- **JavaScript** – Interactive elements and functionality
- **Google Fonts** – Typography
- **SVG** – Logo and graphical elements
